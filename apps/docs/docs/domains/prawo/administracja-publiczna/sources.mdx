Lista aktów prawnych, rozporządzeń, wytycznych i materiałów interpretacyjnych wykorzystywanych w pod‑domenie.

- Rejestry obszarowe:
	- Archiwalno‑kancelaryjne: [obszary/archiwalno-kancelaryjne/zrodla](obszary/archiwalno-kancelaryjne/zrodla/)
	- KIO / PZP: [obszary/kio-pzp/zrodla](obszary/kio-pzp/zrodla/)

## Zasady zapisu
- Zapisuj pełną nazwę aktu + podstawę publikacji (np. Dz.U.) + link + „stan na dzień”.
- Jeżeli przepis ma wiele wersji, dopisz zakres obowiązywania.
- Preferuj linki do oficjalnych publikacji + własne notatki i cytowania.

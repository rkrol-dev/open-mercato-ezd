Wymagania dot. uprawnień: kto może widzieć/edytować, jak ograniczać dane, jakie są role.

Format:
- ID: AP-ACC-001
- Zasób:
- Operacja:
- Kto:
- Warunki (np. organizacja/tenant/sprawa):

Startery:
- AP-ACC-001: Dostęp do akt sprawy ograniczony do osób przypisanych do sprawy + archiwum.
- AP-ACC-002: Eksport pakietu dowodowego dostępny tylko dla uprawnionych ról.

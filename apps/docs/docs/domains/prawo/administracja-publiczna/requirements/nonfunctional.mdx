Wymagania dot. bezpieczeństwa, audytu, retencji, dostępności, wydajności, eksportów.

Startery (do doprecyzowania):
- AP-NFR-001: Ślad audytowy musi obejmować kto/kiedy/co (zmiany metadanych i treści załączników).
- AP-NFR-002: Eksport paczki dowodowej jako spójny pakiet (np. zip) z checksumami.
- AP-NFR-003: Retencja i brakowanie wspierają reguły oparte o kategorię archiwalną.

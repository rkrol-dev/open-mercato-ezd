Aktorzy (role) uczestniczący w procesach, wraz z odpowiedzialnościami i uprawnieniami.

Format:
- Rola:
- Odpowiedzialności:
- Uprawnienia / ograniczenia:
- Dane wrażliwe / zakres dostępu:
- Powiązane procesy:

Startery (do uzupełnienia):
- Pracownik kancelarii / sekretariatu
- Referent sprawy
- Kierownik komórki
- Archiwista / archiwum zakładowe
- Pełnomocnik
- Zamawiający / Wykonawca (PZP)

## Zakres wiedzy do zebrania
- Organizacja kancelarii / instrukcja kancelaryjna (rola, minimalne elementy).
- JRWA / wykaz akt: klasy, hasła, kategorie archiwalne.
- Akta sprawy: kompletowanie, metryka sprawy, spisy spraw.
- Archiwum zakładowe: przekazywanie, przejmowanie, brakowanie.
- Wymagania dotyczące metadanych i śladów audytowych.

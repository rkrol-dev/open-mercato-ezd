---
title: Punkty integracji
sidebar_position: 50
---

Jak domena może integrować się z innymi obszarami systemu (bez relacji ORM między modułami).

Przykłady do rozważenia:
- Powiązania przez ID (np. `caseId`, `appealId`).
- Widget injection w admin UI (np. karta „Akta sprawy” w formularzach).
- Zdarzenia (events) emitowane przy kluczowych zmianach.
